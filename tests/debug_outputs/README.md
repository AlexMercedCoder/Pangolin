# Debug Outputs

This directory contains debug and test output files from various testing sessions.

## Files

- `debug_final_run.txt` - Final debug run output
- `debug_output.txt` - General debug output
- `debug_output_final.txt` - Final debug output
- `debug_output_final_2.txt` - Second final debug output
- `test_output.txt` - Test execution output

## Purpose

These files are kept for historical reference and debugging purposes. They can be safely deleted if no longer needed.

## Cleanup

To remove all debug outputs:
```bash
rm tests/debug_outputs/*.txt
```
