# Utility Scripts

This directory contains utility scripts for documentation and repository maintenance.

## Scripts

### Documentation Utilities

- **`check_links.sh`** - Validates links in documentation files
- **`fix_doc_links.sh`** - Automatically fixes broken documentation links
- **`combine_markdown_pdf.py`** - Combines markdown files and generates PDFs

### Debug Utilities

- **`manual_debug.sh`** - Manual debugging helper script

## Usage

### Check Documentation Links
```bash
./scripts/utilities/check_links.sh
```

### Fix Documentation Links
```bash
./scripts/utilities/fix_doc_links.sh
```

### Combine Markdown to PDF
```bash
python scripts/utilities/combine_markdown_pdf.py
```

### Manual Debug
```bash
./scripts/utilities/manual_debug.sh
```

## Notes

These scripts are for development and maintenance purposes. They are not required for normal Pangolin operation.
